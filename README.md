# MDReader [markdown查看器]

![markdown查看器](doc/ic_launcher.png)

## 主要功能

1. [x] Markdown 解析为 Html （html文件默认存放在与原md文件同一路径下）
2. [x] 支持表格
3. [x] 支持文中图片 （在写Markdown文件时使用相对路径即可）
4. [ ] 增加 css 样式
5. [ ] 其他
